# 🌐 Portfolio Website — Sid Ali Rezzoug

Personal developer portfolio built as a single-file HTML/CSS/JS application. Dark modern aesthetic with smooth scroll animations, animated skill bars, project cards, experience timeline, and a contact section.

🔗 **Live:** [your-portfolio-url.netlify.app](https://your-portfolio-url.netlify.app)

---

## Sections

- **Hero** — animated geometric avatar, availability status, key stats
- **Projects** — featured client projects and personal/academic work
- **Skills** — animated progress bars + categorized tech stack tags
- **Experience** — vertical timeline with internship and freelance roles
- **Clients** — testimonials from real clients
- **Contact** — email, LinkedIn, GitHub links

## Tech

- Pure HTML5 / CSS3 / Vanilla JavaScript — zero dependencies, zero build tools
- Google Fonts (Space Grotesk + JetBrains Mono)
- Deployed on Netlify (drag-and-drop, single file)

## Run Locally

```bash
# Just open the file — no build step needed
open index.html
```

Or serve with any static server:

```bash
npx serve .
```

## Deploy

1. Go to [netlify.com](https://netlify.com)
2. Drag `index.html` onto the deploy area
3. Done — live in seconds
